# Changelog

## 1.0.0 - 2026-06-25

Initial release.

- Added RSS and Atom fetching through `curl`.
- Added bar unread badge with loading and error state.
- Added triage panel with search, filters, pinning, read actions, and feed errors.
- Added settings UI for feeds, categories, keyword rules, refresh behavior, sorting, dedupe, and OPML.
- Added `>news` launcher provider.
- Added IPC commands for refresh, mark-all-read, clear read state, panel toggle, and search.
- Added OPML import/export helpers.
